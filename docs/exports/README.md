# PDF Export Samples

This directory contains sample PDF exports generated by the Noah Planner application.

## Features

The PDF export feature supports:

- **Week View Export**: Export a 7-day week view (Monday to Sunday) with all events
- **Month View Export**: Export a full calendar month with all events
- **Category Colors**: Visual indicators for event categories
- **Legend**: Color legend showing all categories used in the exported period
- **Localized Dates**: German date formats (e.g., "1.–7. März 2024")
- **High Quality**: 300 DPI resolution for crisp printing
- **Inter Font**: Embedded Inter font for consistent typography
- **A4 & Letter**: Support for both A4 and US Letter paper sizes

## Usage

### From Command Palette

1. Press `Ctrl+K` (or `Cmd+K` on Mac) to open the command palette
2. Type "export" to see export options:
   - "Woche als PDF exportieren" - Export current week
   - "Monat als PDF exportieren" - Export current month
3. Select the desired export option
4. Choose a file path in the export dialog
5. Click "Exportieren"

### From Code

```cpp
// Export a week
planner.exportWeekPdf("2024-03-04", "/path/to/wochenplan.pdf");

// Export a month
planner.exportMonthPdf("2024-03-01", "/path/to/monatsplan.pdf");
```

## Technical Details

- **PDF Library**: Qt6 PrintSupport (QPdfWriter)
- **Layout Engine**: Custom QPainter-based rendering
- **Font Embedding**: Inter Regular and Inter Bold
- **Color Space**: RGB for screen and print
- **Page Margins**: 15mm on all sides
- **Content Sections**:
  - Header (20mm): Title and date range
  - Content (flexible): Calendar grid with events
  - Legend (15mm): Category color legend

## Format Details

### Week View
- 7 columns (Monday to Sunday)
- Day headers with date (e.g., "Montag, 04.03.")
- Event boxes with time and title
- Category color indicators
- Up to ~8-10 events per day visible

### Month View
- Calendar grid with day headers
- Date numbers in top-left of each cell
- Up to 3 events per cell (with "+N more" indicator)
- Category color bars on event entries
- Compact layout optimized for full month overview
